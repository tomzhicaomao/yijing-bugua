## 已修复的问题

1. **严格模式冲突**: `text=虚拟摇卦` 匹配到多个元素 → 使用 `p:has-text("虚拟摇卦").first()`
2. **按钮点击被拦截**: 底部导航栏覆盖掷铜钱按钮 → 使用 `evaluate()` 直接点击
3. **导航栏测试**: HOME 在首页是 `<span>` 不是 `<a>` → 使用 `text=HOME` 而非 `a:has-text("HOME")`
4. **筛选标签测试**: 文本重复 → 使用 `.first()` 选择器
5. **统计页面测试**: Mock 数据未生效 → 支持空状态和有数据两种情况
6. **页面过渡测试**: PageTransition 保留 DOM 元素 → 检查新步骤可见而非旧步骤隐藏
