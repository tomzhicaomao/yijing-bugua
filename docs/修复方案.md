## 三、修复方案明细

### F1 — Coin 虚拟摇卦架构+概率双重错误 🔴

#### 问题分析

**错误 1 — 架构错误**:

```typescript
// DivineView.tsx 第 258-266 行
{method === 'virtual' && currentIndex < 6 && (
  <div className="flex justify-center gap-6 py-8">
    {[0, 1, 2].map((i) => (
      <Coin key={i} onFlip={(backs) => {
        const value = backs === 0 ? 6 : backs === 1 ? 7 : backs === 2 ? 8 : 9
        setLineValue(value)  // 每个 Coin 独立生成一爻
      }} />
    ))}
  </div>
)}
```

3 个 Coin 各被点击一次 → 调用 `setLineValue` 3 次 → 设置 3 个爻值。这不是"掷三枚铜钱定一爻"，而是"掷三枚铜钱定三爻"。

**正确行为**: 用户一次操作 → 3 枚铜钱同时抛掷 → 计算背面总数（0-3）→ 映射为 1 个爻值。

**错误 2 — 概率错误**:

```typescript
// Coin.tsx 第 17 行
const backs = Math.floor(Math.random() * 4)  // 0,1,2,3 各 25%
```

本应为：P(0)=12.5%, P(1)=37.5%, P(2)=37.5%, P(3)=12.5%

#### 修复步骤

**方案 A（推荐：替换为 VirtualCoins）**：

```diff
// DivineView.tsx 第 9 行
- import Coin from '../components/casting/Coin'
+ import VirtualCoins from '../components/casting/VirtualCoins'

// 第 258-266 行
- {method === 'virtual' && currentIndex < 6 && (
-   <div className="flex justify-center gap-6 py-8">
-     {[0, 1, 2].map((i) => (
-       <Coin key={i} onFlip={(backs) => {
-         const value: 6 | 7 | 8 | 9 = backs === 0 ? 6 : backs === 1 ? 7 : backs === 2 ? 8 : 9
-         setLineValue(value)
-       }} />
-     ))}
-   </div>
- )}
+ {method === 'virtual' && (
+   <VirtualCoins currentIndex={currentIndex} onCast={setLineValue} />
+ )}
```

**验证方法**:
```bash

npx vitest run tests/engine/casting.test.ts

# 确认 VirtualCoins 调用 tossCoinsDetailed

# 预期：第 28 行调用
```

### F3 — 6 爻填满后流程可靠性缺陷 🔴

#### 问题分析

当前完成 6 爻的逻辑：

```typescript
// DivineView.tsx 第 37-42 行
const completingRef = useRef(false)
useEffect(() => {
  if (step === 'casting' && currentIndex >= 6 && !completingRef.current) {
    completingRef.current = true
    completeCasting().finally(() => { completingRef.current = false })
  }
}, [step, currentIndex, completeCasting])
```

依赖 `useEffect` + ref 防重入的模式在 React 18/19 严格模式下可能触发多次，且 `completeCasting` 是异步的（调用 `createRecord` + `navigate`），防重入 ref 的 timing 可能被异步流程打乱。

#### 修复步骤

```diff
// useDivination.ts — setLineValue 函数内
const setLineValue = useCallback((value: LineValue) => {
  if (currentIndex >= 6) return
  // ...
  newLines[currentIndex] = value
  setLines(newLines)
- setCurrentIndex(currentIndex + 1)
+ const newIndex = currentIndex + 1
+ setCurrentIndex(newIndex)
+ 
+ // 第 6 爻填满后自动触发完成
+ if (newIndex >= 6) {
+   // 使用微任务确保状态更新后再触发
+   queueMicrotask(() => completeCasting())
+ }
}, [currentIndex, lines, castingTimestamp, completeCasting])
```

