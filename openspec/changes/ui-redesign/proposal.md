# UI Redesign - 混合风格重新设计

## 概述

为易经占卜应用进行全面的 UI 重新设计，采用混合风格（传统元素 + 现代交互），支持移动端，添加动画效果。

## 设计目标

1. **混合风格**：传统中国美学 + 现代 UI 设计
2. **全部页面重做**：7 个页面全部重新设计
3. **移动端支持**：响应式设计，优化移动端体验
4. **动画效果**：流畅的过渡动画和交互反馈

## 设计原则

### 传统元素
- 中国书法字体
- 水墨画风格背景
- 传统图案（云纹、回纹等）
- 自然色彩（墨色、朱砂、青铜、玉石）

### 现代交互
- 清晰的视觉层次
- 流畅的动画过渡
- 直观的操作反馈
- 良好的可访问性

## 页面范围

1. **HomeView** - 首页
2. **Login/Register** - 登录/注册
3. **DivineView** - 起卦页面
4. **ResultView** - 结果页面
5. **HistoryView** - 历史列表
6. **HistoryDetailView** - 历史详情
7. **StatsView** - 统计页面
8. **SettingsView** - 设置页面

## 技术约束

- 保持 React + TypeScript + Tailwind CSS
- 使用 Framer Motion 实现动画
- 支持 iOS Safari 和 Android Chrome
- 保持现有的业务逻辑不变

## 设计参考

- 传统中国风 App（如：故宫、禅意应用）
- 现代极简设计（如：Notion、Linear）
- 混合风格案例（如：冥想应用、茶道应用）

## 验收标准

1. 所有页面在移动端（375px）和桌面端（1440px）都能正常显示
2. 关键操作有流畅的动画过渡
3. 视觉风格统一且符合"混合风格"定位
4. 可访问性符合 WCAG 2.1 AA 标准
5. 性能指标：LCP < 2.5s, FID < 100ms
