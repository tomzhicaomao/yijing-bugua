```css
}
```

## 动画设计

### 页面过渡

```typescript
// 页面切换动画
const pageVariants = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  exit: { opacity: 0, y: -20 }
}

// 铜钱投掷动画
const coinVariants = {
  idle: { rotateY: 0 },
  flipping: { rotateY: 180, transition: { duration: 0.5 } },
  landed: { rotateY: 360, transition: { duration: 0.3 } }
}
```

## 实现阶段

### 阶段 1: 基础设施 (1 天)

- 安装 Framer Motion
- 创建设计令牌 (Design Tokens)
- 更新 Tailwind 配置
- 创建基础组件库

### 阶段 2: 核心组件 (2 天)

- 卡片组件
- 按钮组件
- 输入组件
- 导航组件

### 阶段 3: 页面重设计 (3-4 天)

- 首页
- 登录/注册
- 起卦页面
- 结果页面
- 历史页面
- 统计页面
- 设置页面

### 阶段 4: 动画与优化 (1-2 天)

- 页面过渡动画
- 微交互动画
- 性能优化
- 移动端测试

**总计: 7-9 天**
